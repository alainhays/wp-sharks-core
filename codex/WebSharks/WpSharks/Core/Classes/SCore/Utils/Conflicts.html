protected maybeNotify()

Internal Note

Intentionally choosing not to use built-in notice utilities here. The notice utilities set option values, and if we are in conflict with another application (e.g., lite/pro edition) that could lead to unforeseen problems.
Not only that, but the hooks needed to use notice utilities are not attached until after a check for conflicts has been finalized; i.e., notice utils won't work anyway.
